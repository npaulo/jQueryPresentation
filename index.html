The Document Ready EventÂ¶

This is to prevent any jQuery code from running before the document is finished loading (is ready).

Long form:
$(document).ready(function(){
   // jQuery methods go here...
});

Short form:
$(function(){
   // jQuery methods go here...
});

Basic SyntaxÂ¶

$("selector").action()
  • A $ sign to define/access jQuery
  • A (selector) to "query (or find)" HTML elements
  • A jQuery action() to be performed on the element(s)

Basic SelectorsÂ¶

Syntax Description 
$("*")
 Select all elements. 
$(".class-name")
 Selects all elements with the given css class. 
$("span")
 Selects all elements with the given tag name. 
$("#id")
 Selects a single element with the given id attribute.

Selects the combined results of all the specified selectors.

Hierarchy SelectorsÂ¶

Syntax Description 
$(".content > b")
 Selects all direct child elements specified by “b” (child) of elements specified by “.content” (parent). 
$("ul label")
 Selects all elements that are descendants of a given ancestor. 
$("ul[data-list=first] + b")
 Selects all next elements matching “b” (next) that are immediately preceded by a sibling “ul[data-list=first]” (prev). 
$("ul[data-list=first] ~ ul")
 Selects all sibling elements that follow after the “ul[data-list=first]” (prev) element, have the same parent, and match the filtering “ul” (siblings) selector.

HTML Content and AttributesÂ¶

Four simple, but useful, jQuery methods for DOM manipulation are:

  • .text() - Gets or Sets the text content of selected elements
  • .html() - Gets or Sets the content of selected elements (including HTML markup)
  • .val() - Gets or Sets the value of form fields
  • .attr() - Gets or Sets the attribute value
Get value:console.log("Text: " + $("#test").text());
Set value:$("#test").text("Set this text!!!");

Add New HTML ContentÂ¶

We will look at four jQuery methods that are used to add new content:
  • .append() - Inserts content at the end of the selected elements
  • .prepend() - Inserts content at the beginning of the selected elements
  • .after() - Inserts content after the selected elements
  • .before() - Inserts content before the selected elements

Remove Elements/ContentÂ¶

To remove elements and content, there are mainly two jQuery methods:
  • .remove() - Removes the selected element (and its child elements)
  • .empty() - Removes the child elements from the selected element

Manipulating CSSÂ¶

jQuery has several methods for CSS manipulation. We will look at the following methods:
  • .addClass() - Adds one or more classes to the selected elements
  • .removeClass() - Removes one or more classes from the selected elements
  • .toggleClass() - Toggles between adding/removing classes from the selected elements
  • .hasClass() - Determine whether any of the matched elements are assigned the given class
  • .css() - Sets or returns the style attribute

Events other syntaxÂ¶

The other syntax

  • .on() - Attach an event handler function for one or more events to the selected elements;
  • .off() - Remove an event handler;
  • .trigger( eventType ) - Execute all handlers attached to the matched elements for the given event type.

AJAX - Asynchronous JavaScript and XMLÂ¶

AJAX is about loading data in the background and display it on the webpage, without reloading the whole page.
  • .load() - The method loads data from a server and puts the returned data into the selected element;
  • .get() - Requests data from a specified resource;
  • .post() - Submits data to be processed to a specified resource;
  • .ajax() - Generic ajax request;
